Linux常用命令
发表于 2016-08-06| 更新于 2020-06-28|Linux
字数总计: 215|阅读时长: 1 分钟| 阅读量:

守护进程启动spring-boot项目:

nohup java -jar repair-api.jar > log.file 2>&1 &

实时查看日志信息(接上):

tail -f log.file

查看某个端口的进程id:

lsof -i:8080

通过进程id 关闭某个端口(注:中括号请忽略):

kill -9 [从上面获取的进程id]

创建mysql utf8mb4数据库

create database zctqgl default character set utf8mb4 collate utf8mb4_unicode_ci;

进入mysql docker容器内

docker exec -it mysql bash

复制.sql文件到docker容器内

docker cp xxx.sql mysql:/zcjksb.sql

ls -l xxx.sql

ssh连接服务器

ssh xxx@192.168.0.124

scp传输文件

scp -P 1070 xxx.jar xxx@xx.x.xx.xxx:~/

移动当前文件夹内容至上一级目录

mv * ../

删除名为 RUNOOB 的数据库

mysql> drop database RUNOOB;

查询某张表里最新后一条数据

mysql:

select * from table order by id DESC limit 1

oracle:

select * from emp where id in (select max(id) from emp);
