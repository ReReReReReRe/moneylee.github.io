###Spring5的简单学习

什么是注解：

  • Java注解是附加在代码中的一些元信息， 用于一些工具在编译，运行时进行解析和使用，起到说明配置的功能
  • 注解本质上继承 Annotation接口，我们可以通过反射获取注解的相关信息，从而做些逻辑操作
  • springboot里面大量使用了注解，@Controller、 @RestController、 @service、 @Autowire 等

什么是Spring框架：

  • 轻量级的 DI / IoC 和 AOP容器的开源框架

  • 有啥好处：

    • 管理创建和组装对象之间的依赖关系， 加了Spring注解的类会自动创建一个实例，加到IOC容器里面，然后哪里需要它，就自动赋值过去

      • 使用前： 手工创建

      • 使用后：Spring创建， 自动注入

SpringBoot2.x和SpringCloud微服务的关系

单体应用 微服务
开发速度慢 易开发、理解和维护
启动时间长 独立的部署和启动
依赖庞大 问题：分布式系统的（分布式事务问题）、需要多个服务（服务治理）

SpringCloud基于SpringBoot 多了几个组件 多了几个依赖和注解 注册中心 网关 服务 鉴权 等等很多模块 例如注册中心可以理解为一个项目 是基于SpringBoot的

SpringBoot常用注解

@Controller

作用：用于标记这个类是控制器，返回页面的时候使用，如果要返回JSON，则需要在接口上使用@ResponseBody才可以

####@RestController

作用：用于标记一个类是控制器，返回JSON数据的时候使用，如果使用这个注释，则接口返回数据会被序列化为JSON

所以：@RestController = @Controller + @ResponseBody

@RequestMapping

作用：路由映射，用于类上做1级映射；用于某个方法上做子路径

@SpringBootApplication

作用：用于标记是SpringBoot应用，里面包含多个子注解，即

SpringBoot2.x内部定制JSON字段

  • 常用框架 阿里 fastjson 谷歌gson
  • JavaBean序列化为JSON
    • 性能：jackjson > fastjson > gson > json-lib 同一个结构
    • jackon、fastjson、gson类库各有所长，各有自己的专长
    • 空间换时间、时间换空间
  • Jackson处理相关
    • 指定字段不返回：@JsonIgnore
    • 指定日期格式：@JsonFormat(pattern=”yyyy-MM-dd hh:mm:ss”, local=”zh”, timezone=:”GMT+8”)
    • 空字段不反回：@JsonInclude(Include.NON_NULL)
    • 指定别名：@JsonProperty
  • 开发功能：视频创建时间返回自定义格式；过滤用户敏感信息

      • 1、添加 @Component注解；

      • 2、使用 @PropertySoure 注解指定配置文件位置；

      • 3、使用 @ConfigurationProperties 注解，设置相关属性；
